 中国戏曲文化传播与电商平台 (Chinese Opera Culture & E-commerce Platform)

> 弘扬传统文化 · 全栈开发实践 · 响应式设计

项目简介
本项目是一个基于 **Java Web (Servlet)** 技术栈构建的综合性平台，旨在通过数字化手段推广中国戏曲文化。项目集成了**文化科普**（行当、剧种）与**文创电商**（商品浏览、购物车、订单管理）两大核心功能。
系统采用前后端分离的设计思想（逻辑层与视图层解耦），实现了从用户注册登录、商品交易到后台数据可视化的完整业务闭环。

核心亮点
沉浸式国风设计：
  - 摒弃通用模板，定制“胭脂红”与“千岁绿”传统配色方案。
  - 纯手写 **CSS3 动画**实现“手风琴”剧种折叠、图片悬停放大及无缝轮播效果，无依赖重型前端框架。
  - 使用 **Photoshop** 自制高清戏曲海报与背景纹样，解决素材版权与风格统一难题。
  
稳健的后端架构：
  - 基于 **Java Servlet** 构建 MVC 架构，利用 **Filter** 实现全局字符编码处理与用户登录权限拦截。
  - 使用 **JDBC** 连接 **MySQL** 数据库，设计了规范的用户、商品、订单及戏曲资源表结构，支持复杂的多表关联查询。
  - 集成 **Thymeleaf** 模板引擎，实现服务端动态渲染，提升首屏加载速度与 SEO 友好度。

工程化实践与协作：
  - **代码重构**：将初期的 Table 布局全面重构为 **Flex/Grid + Div** 响应式布局，显著提升代码可维护性与多端适配能力。
  - **团队协作**：三人小组分工明确（后端逻辑/前端交互/UI设计），通过统一命名规范与模块化开发，有效解决了代码合并冲突问题。
  - **智能辅助**：在开发复杂 JS 交互（如动态宽度计算）时，结合 AI 工具进行逻辑分析与 Bug 排查，提升了开发效率。

🚀 功能模块

用户端 (Front-end)
- **首页门户**：自定义导航栏、动态轮播图、四大行当展示、五大剧种手风琴特效。
- **用户中心**：注册/登录（含邮箱验证码）、收藏夹管理。
- **文创商城**：商品列表检索、购物车增删改查、模拟下单流程。
- **文化百科**：戏曲发展历程、名家介绍、学术资源展示。

管理员端 (Back-end)
- **数据看板**：可视化展示周访问量、用户增长趋势及热门商品统计。
- **资源管理**：对戏曲剧目、新闻公告、文创商品进行完整的 **CRUD** (增删改查) 操作。
- **订单与用户**：查看订单详情、管理注册用户权限。
